# Screen

## About:

#### Screen is a Spigot 1.16-1.16.2 Plugin that mirrors your display enabling you to do things such as playing minecraft within minecraft. (NOTE: Screen is read-only and it is therefore highly reccomened that the user has multiple monitors.)

## Features:

* Display Screen live into Minecraft World
* Multi-Screen Support
* Custom Resolution
* Custom Location
* Linux and Windows Support
* Actively Developed and Maintained

## Setup

(Required) [Wiki](https://github.com/carter-0/screen/wiki/)

## Dependencies

* Minecraft 1.16 - 1.16.2 Spigot Server
* (Linux Only) ImageMagick
